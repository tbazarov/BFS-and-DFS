# BFS-and-DFS
Репозиторий для программы GraphTraversal
Программа реализует алгоритм обхода графа: DFS(поиск в глубину) и BFS(gjbcr d ibhbye)
Используется паттерн проектирования "Шаблонный метод" и алгоритмы имеют общую структуру - паттерн "Стратегия"
Как запускать программы:
1) mkdir build && cd build
2) cmake .. && make
3) ./GraphTraversal
